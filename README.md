# FiveSheep Skills
> Agent skills for iOS app development.

## Skills

### `code-review`
> Review code according to FiveSheep best practices.

This skill comes with two modes:
- **Passive**: Light automatic code review after any significant change.
- **Manual**: Request a thorough code review, for the whole codebase or a specific part of it.

The passive mode is always active and should run automatically when necessary.

**Install**: `npx skills add https://github.com/FiveSheepCo/skills --skill code-review`
